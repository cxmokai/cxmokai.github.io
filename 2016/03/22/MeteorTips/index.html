1. JavaScript程序调试

How to debug a basic node.js application

使用node-inspector可以调试node程序, 步骤如下：

  1. 全局安装node-inspector

  2. 在终端运行node-inspector

  3. 打开浏览器(Chrome或者Firefox)输入http://localhost:8080/debug?port=5858 。你可以看到node-inspector的图形界面，但是现在还没有在debug的程序。

  4. 在另一个终端以debug模式运行你的程序，比如：node –debug test.js
  5. 刷新刚才的node-inspector页面，开始调试。
